# 整数

`num` 型では，8バイトの符号付き整数を扱えます．`num` 型同士で，以下のような演算を行えます．

## 2項演算

### Assignment

代入演算は `:` で行います．

```
num a, b

a: 10
b: 20
a: b
a: b: 100
```

6行目では， `b` に `100` が代入された後， `a` にも `100` が代入されます．

### Additive

和は `+` ，差は `-` で行います．

```
num a, b, c

a: 10
b: 20
c: 1 + 2
c: a + b
c: a - 200
```

`c` には，5行目では `3` ，6行目では `30` ，7行目では `-190` が代入されます．
